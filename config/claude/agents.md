
## Critical Rules
- **NEVER GIVE UP**: When facing compilation errors or private import issues, work through them systematically
- **ASK FOR HELP**: When you encounter what seems like an unsolvable problem, ASK THE USER before abandoning the approach
- **NO SIMPLE SOLUTIONS**: Don't take the easy way out. If the user wanted the simple solution, they would have asked for it
- **FINISH WHAT YOU START**: Complete the task you set out to do, don't switch approaches midway
